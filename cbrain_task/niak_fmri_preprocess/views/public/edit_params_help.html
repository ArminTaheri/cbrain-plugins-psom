Niak_fmri_preprocess 0.13.4


fMRI pre-processing pipeline as available in Niak. More information at https://github.com/SIMEXP/niak

Parameters

  • Output folder name (--folder_out) : The name of the output folder where results will be produced.
  • PSOM max queued (--opt-psom-max_queued) : Number of threads to use.
  • Slice timing type scanner (--opt-slice_timing-type_scanner)
  • Slice timing type acquisition (--opt-slice_timing-type_acquisition)
  • Slice timing delay in tr (--opt-slice_timing-delay_in_tr)
  • Resample vol voxel size (--opt-resample_vol-voxel_size) : The voxel size to use in the stereotaxic space.
  • T1 preprocess nu correct arg (--opt-t1_preprocess-nu_correct-arg) : Parameter for non-uniformity correction. 200 is a suggested value for 1.5T images, 75 for 3T images.
  • Time filter hp (--opt-time_filter-hp) : Cut-off frequency for high-pass filtering, or removal of low frequencies (in Hz).
  • Time filter lp (--opt-time_filter-lp) : Cut-off frequency for low-pass filtering, or removal of high frequencies (in Hz).
  • Regress confounds flag gsc (--opt-regress_confounds-flag_gsc) : Global signal regression.
  • Regress confounds flag scrubbing. (--opt-regress_confounds-flag_scrubbing)
  • Regress confounds thre fd (--opt-regress_confounds-thre_fd) : The threshold on frame displacement for scrubbing
  • Smooth vol fwhm (--opt-smooth_vol-fwhm) : Full-width at maximum (FWHM) of the Gaussian blurring kernel, in mm.
  • Extra arguments : Extra arguments passed to niak_cmd

Input files

  • Input files (--file_in) : A directory containing input files.

Output files

  • Output folder : A folder containing the output files.
